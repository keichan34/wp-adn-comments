ADN Comments is a plugin to replace native WordPress comments with a thread of App.net posts.

== Description ==

ADN Comments replaces the native WordPress comments with the App.net Coments Widget.

For more information, please see the [blog post](http://blog.app.net/2014/02/13/what-is-the-app-net-comments-widget/) about the App.net Comments Widget.

If you want to set your username as the default @-reply for comments, enter your App.net username in your profile settings.

== Installation ==

1. Upload the plugin directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
